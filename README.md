# AI ML Journey Tracker

A modern, aesthetic, responsive productivity web app for managing your complete AI/ML learning journey.

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Local Storage persistence
- React Hot Toast
- React Confetti

## Features

- Startup-style glassmorphism dashboard
- AI/ML roadmap tracking with progress updates
- Smart to-do list (add/edit/delete/complete/filter/search/priority/due dates)
- Notes manager with search and inline editing
- Daily consistency tracker with streak and weekly chart
- Goal manager for short/long-term targets
- Pomodoro timer and motivational quote section
- Dark/Light mode toggle

## Setup

```bash
npm install
npm run dev
```

Then open the local URL shown in terminal (usually `http://localhost:5173`).
